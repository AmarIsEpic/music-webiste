A Metallica-inspired web application for selecting your favorite songs.
Description
MUSICA is a dark-themed music selection interface featuring Metallica's discography. Users can browse through 12 studio albums, select their favorite songs, and prepare for a personality analysis based on their choices.
Features

Browse 12 Metallica albums with cover art
Select songs from each album
Real-time selection counter
Responsive design for all devices
Custom Metallica-style typography and dark aesthetic

Getting Started

Make sure all files are in the same folder:

index.html
styles.css
app.js
data.js
mfont.ttf (font file)
Images/ folder with album covers


Open index.html in your web browser

How to Use

Click on any album to view its songs
Click on songs to select/deselect them
Use the "BACK" button to return to albums
The "CONTINUE" button activates when you select at least one song

Technologies

HTML5
